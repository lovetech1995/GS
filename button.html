📚 #3.Button & Draw Button


👀 Button trông nó như thế nào ?

Button là nút ấn để kích hoạt đoạn script bên ngoài Editor, và trông nó đơn giản như vầy


Ngoài ra, chúng ta có thể vẽ một button nằm ngoài WorkSheet, thì nó sẽ là Draw Button


🔴 Tạo Button như thế nào ??

Giờ mình tạo một button in ra chữ Hello World để làm ví dụ nhé


Đề bài như sau:

⏱ Tạo Button kích hoạt script trong Code Editor với hành động in ra một dòng chữ Hello World

🌈 Để tạo button trên thanh Tool-Bar bạn hãy làm theo những bước sau

  • Bước 1: Tạo một project script 👉 Đọc bài Project Script
  • Bước 2: Nhập dòng code bên dưới vào khung script editor, với giải thích như bên dưới!

    • // Function thứ 1 là khởi tạo nút trên thanh tool-bar
    function onOpen(e){
          var ui = SpreadsheetApp.getUi(); // ---> Truy xuất vào giao diện user interface của worksheet
          var Menu = ui.createMenu('Pop Up'); // ---> Tạo một Menu với tên là Pop Up
          Menu.addItem('Say Hello World','Popup').addToUi(); //--->Tạo một thao tác với tên là Say Hello World, khi nhấn vào thao tác sẽ kích script Pop up
    };
            
    // Function thứ 2 là đoạn script kích hoạt cửa sổ thông báo trên Worksheet 
        
    function Popup() {   
          Browser.msgBox('hello world'); // ---> Mở bảng thông báo với dòng chữ hello world
    };
        
    }

  • Bước 3: Lưu dòng code này lại, và click vào nút RUN


  • Bước 4: Trên thanh công cụ xuất hiện nút PopUp, chúng ta click chọn để kích hoạt đoạn script. Kết quả như hình


🌈 Để tạo button trên thanh WorkSheet, các bạn làm như sau

  • Bước 1,2: Làm tương tự như bước ở trên
  • Bước 3: Chúng ta chọn vào công cụ vẽ ở ngoài tool-bar bằng cách click Insert > Drawing, sau đó chọn công cụ vẽ hình chữ nhật để vẽ cho mình một nút nhấn


  • Bước 4: Sau khi vẽ xong click phải vào hình chữ nhật chọn Edit Text, điền Pop Up và căn chỉnh định dạng (Ở đây các thao tác gần như giống Doc, nên rất đơn giản, bạn cứ làm thôi nhé! ). Sau khi xong nhấn Save and Close chúng ta sẽ có một cái nút Pop Up như hình


  • Bước 4: Click chuột phải chọn Assign Script, sau đó gõ đoạn tên hàm Pop Up Editor Code.


  • Bước 5: Nhấn vào Nút vừa tạo để tận hưởng thành quả như trên

👉 Đã xong rồi đó, bài này chủ yếu là để cho các bạn hiểu về nút và cách ứng dụng của nó. Hãy đón xem bài hướng dẫn tiếp theo nhé !
